Why Semantic HTML Matters

How using correct HTML tags improves SEO and accessibility.
Accessibility ♿

Screen readers used by blind/low-vision users rely on semantic tags. nav = "this is navigation". main = "this is the main content". If you use 20 divs, a screen reader user gets lost. With semantic HTML, they can skip straight to content.
